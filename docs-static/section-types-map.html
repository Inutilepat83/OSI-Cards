#Map Section

Displays geographic data with embedded maps, pins, and location information.

#Overview

The Map Section (type: "map") is used for displays geographic data with embedded maps, pins, and location information.

#Quick Facts

PropertyValue
Typemap
Uses FieldsYes
Uses ItemsNo
Default Columns1
Supports CollapseNo
Aliaseslocations

#Use Cases

  • Office locations
  • Store finder
  • Geographic data
  • Location tracking

json
{
  "title": "Global Presence",
  "type": "map",
  "description": "Office locations worldwide",
  "fields": [
    {
      "name": "Headquarters",
      "x": 37.7749,
      "y": -122.4194,
      "type": "office",
      "address": "San Francisco, CA, USA"
    },
    {
      "name": "European HQ",
      "x": 51.5074,
      "y": -0.1278,
      "type": "office",
      "address": "London, UK"
    },
    {
      "name": "APAC Office",
      "x": 35.6762,
      "y": 139.6503,
      "type": "branch",
      "address": "Tokyo, Japan"
    }
  ]
}
