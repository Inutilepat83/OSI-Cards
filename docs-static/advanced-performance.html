#Performance Optimization

Optimize OSI Cards for best performance.

#Strategies

#1. Virtual Scrolling

For many cards, use virtual scrolling:

html
&lt;cdk-virtual-scroll-viewport itemSize="400"&gt;
  &lt;app-ai-card-renderer
    *cdkVirtualFor="let card of cards"
    [cardConfig]="card"&gt;
  &lt;/app-ai-card-renderer&gt;
&lt;/cdk-virtual-scroll-viewport&gt;

#2. OnPush Change Detection

All library components use OnPush.

#3. Web Workers

Layout calculations use Web Workers:

typescript
layoutWorker.calculateLayout(cards, width);

#4. Lazy Loading

Lazy load card routes:

typescript
{
  path: 'cards',
  loadComponent: () =&gt; import('./cards.component')
}

#5. Image Optimization

  • Use lazy loading for images
  • Provide appropriate sizes
  • Use modern formats (WebP)

#Metrics to Monitor

  • First Contentful Paint
  • Largest Contentful Paint
  • Time to Interactive
  • Bundle size
